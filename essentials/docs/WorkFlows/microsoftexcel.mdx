Automate your customer engagement by connecting **Microsoft Excel** to **Contactswing**. This guide shows you how to trigger a **phone call** in Contactswing automatically whenever a new row is added in Microsoft Excel, so you can deliver instant, personalized responses.

## Step-by-step guide

### 1. Set up your Microsoft Excel trigger üìù

- **Trigger:** New Row
- **Connection:** `Microsoft-Excel`
- **Worksheet:** Select the worksheet where you‚Äôll be adding new rows.
- Every time you add a new row, this trigger will activate the flow.

---

### 2. Configure the Contactswing Call action üìû

- In Contactswing, choose **CALL** as the response type.
- Enter your **Workspace API Key**.
- Add the **Webhook URL** provided by Contactswing.
- Whenever a new row is added in Excel, Contactswing will automatically place a phone call using the configured details.

---

### 3. Activate your Agentic AI Flow ‚ö°

- Review all trigger and action settings.
- Click **Publish** to activate the flow.
- Once published, the system will automatically trigger the phone call every time a new row is added in Microsoft Excel.
