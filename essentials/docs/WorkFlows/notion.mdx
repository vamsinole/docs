## Step-by-step guide

### 1. Set up your Notion trigger üìù

- **Trigger:** New Database Item
- **Connection:** `Notion`
- **Database:** Select the database where you‚Äôll be adding new items.
- Every time you add a new item, this trigger will activate the flow.

---

### 2. Configure the Contactswing Call action üìû

- In Contactswing, choose **CALL** as the response type.
- Enter your **Workspace API Key**.
- Add the **Webhook URL** provided by Contactswing.
- Whenever a new item is added in Notion, Contactswing will automatically place a phone call using the configured details.

---

### 3. Activate your Agentic AI Flow ‚ö°

- Review all trigger and action settings.
- Click **Publish** to activate the flow.
- Once published, the system will automatically trigger the phone call every time a new item is added in Notion.
