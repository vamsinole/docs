# Aitable to Contactswing Phone Call

Automate your customer engagement by connecting **Aitable** to **Contactswing**. This guide shows you how to trigger a **phone call** in Contactswing automatically whenever a new or updated record is added in Aitable, so you can deliver instant, personalized responses.

## Step-by-step guide

### 1. Create an Aitable connection and generate API token üìù

- Log in to your Aitable account.
- Go to **Account Settings** or **Integrations** and generate a **Personal Access Token**.
- Copy this token.

### 2. Add your Aitable API key in Contactswing üîë

- Open Contactswing dashboard and go to **Connections**.
- Click **Add Connection** and select **Aitable**.
- Paste your generated **Personal Access Token** here.
- Save the connection.

### 3. Select your Aitable workspace (Space) and datasheet (Table) üìã

- In Contactswing, select the Aitable **Space** where you will be monitoring records.
- Choose the **Datasheet** (table) within that space.
- Specify the **Trigger Field** to determine when the flow should activate.
- Select the appropriate **View** (Grid or Gallery) if applicable.

### 4. Configure the Contactswing Call action üìû

- Choose **CALL** as the response type.
- Enter your **Workspace API Key** from Contactswing.
- Paste the **Webhook URL** provided by Contactswing.
- This setup will trigger an automatic phone call whenever a new or updated record matches your trigger criteria.

### 5. Activate your Agentic AI Flow ‚ö°

- Review your settings and connections.
- Click **Publish** to activate the flow.
- Once published, every new or updated Aitable record will trigger a phone call automatically.

## Example scenario

- A new customer record is added or updated in Aitable.
- This event triggers the Agentic AI flow.
- Contactswing automatically initiates a call to the customer with the configured script or agent interaction.
