## Step-by-step guide

### 1. Set up your Airtable trigger üìù

- **Trigger:** New or updated Record

- **Table:** Select the table where you‚Äôll be adding or updating records.

### 2. Configure the Contactswing Call action üìû

- In Contactswing, choose **CALL** as the response type.
- Enter your **Workspace API Key**.
- Add the **Webhook URL** provided by Contactswing.
- Whenever a new or updated record is added in Airtable, Contactswing will automatically place a phone call using the configured details.

---

### 3. Activate your Agentic AI Flow ‚ö°

- Review all trigger and action settings.
- Click **Publish** to activate the flow.
- Once published, the system will automatically trigger the phone call whenever a new or updated record is added in Airtable.
