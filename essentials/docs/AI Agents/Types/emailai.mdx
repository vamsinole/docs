## Step-by-step guide

Follow these steps to create your Email AI Agent:

### Step 1: Access the AI Agent dashboard

Navigate to the **AI Agent** tab and click **New AI Agent** to begin creating your email automation agent.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/1ea.jpg" alt="Add FAQ" />


### Step 2: Choose Email agent type

Select **Email** from the agent type options to create an AI-powered email assistant.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/22.jpg" alt="Add FAQ" />


### Step 3: Configure your agent settings

Set up your agent with these essential configurations:

- **Name**: Give your agent a descriptive name
- **AI Model**: Select your preferred AI model
- **Email Address**: Enter the email address for your agent
- **SMTP Settings**: Configure your email server details
- **Prompt**: Define how your agent should handle emails
- **Actions**: Set up automated actions and responses

Finalize your setup by:
- Testing email connectivity
- Setting response templates
- Configuring auto-reply rules
- Setting working hours
- Defining escalation paths

### Step 4: Define your agent's prompt

Create a detailed prompt that guides your agent's behavior:
- Set the tone and style
- Define handling priorities
- Specify response formats
- Include key information sources
- Set boundaries and limitations
