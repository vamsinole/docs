## Step-by-step guide

Follow these steps to create your WhatsApp AI Agent:

### 1. Access the AI Agent dashboard

Go to the **AI Agent** tab and select **New AI Agent** to start creating your WhatsApp automation agent. Make sure your WhatsApp integration is active in the **Integrations** tab.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/Create-Assistant.jpg" alt="AI Agent dashboard" />

### 2. Choose WhatsApp agent type

Select **WhatsApp** from the agent type options to automate your WhatsApp messaging workflows.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/wp1.jpg" alt="Select WhatsApp Agent type" />

### 3. Configure your agent settings

Name your agent and set up the following options:

- **Name**: Enter a descriptive name (e.g., "My WhatsApp Agent").
- **AI Agent Type**: Choose **Outbound** to send automated WhatsApp messages.
- **Phone Number**: Select the WhatsApp-enabled number for your agent.
- **Check WhatsApp**: Specify how many days your agent should monitor incoming WhatsApp messages.

Click **Next** when you finish configuring your agent.
