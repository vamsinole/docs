## Step-by-step guide

Follow these steps to create your Text AI Agent:

### 1. Access the AI Agent dashboard

Go to the **AI Agent** tab and select **New AI Agent** to start creating your text automation agent.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/1.jpg" alt="AI Agent dashboard" />

### 2. Choose Text agent type

Select **Text** from the agent type options to build a bot for SMS-based interactions and intelligent responses.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/37.jpg" alt="AI Agent dashboard" />

### 3. Configure your agent settings

Name your agent and set up the following options:

- **Name**: Enter a descriptive name (e.g., "My Text AI Agent")
- **AI Model**: Choose an AI model for generating responses
- **Action**: Define actions your agent can trigger during conversations
- **Prompt**: Enter instructions to guide your agent's behavior

Click **Next** when you finish configuring your agent.
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/38.jpg" alt="AI Agent dashboard" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/4.jpg" alt="AI Agent dashboard" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/5.jpg" alt="AI Agent dashboard" />

### 4. Test your agent

Send a test message to ensure your Text AI Agent:

- Responds appropriately to various inputs
- Follows the defined prompt guidelines
- Handles different scenarios correctly
- Maintains natural conversation flow
