## Step-by-step guide

Follow these steps to create your Voice AI Agent:

### 1. Access the AI Agent dashboard

Go to the **AI Agent** tab and select **New AI Agent** to start creating your voice automation agent.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/Create-Assistant.jpg" alt="AI Agent dashboard" />


### 2. Choose Voice agent type

Select **Voice** from the agent type options to build a bot for voice-based interactions and intelligent responses.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/2.jpg" alt="AI Agent dashboard" />


### 3. Configure your agent settings

Name your agent and set up the following options:

- **Name**: Enter a descriptive name (e.g., "My Voice AI Agent").
- **Voice Type**: Select Voice Type (e.g., Inbound or Outbound).
- **Version**: Select V1 or V2. 
    ##### Note: V2 offers access to a wider variety of AI models and improved connectivity speed.
- **Phone Number**: Select the phone number for your agent.
- **AI Model**: Choose an AI model for generating responses.
- **Action**: Define actions your agent can trigger during calls.
- **Prompt**: Enter instructions to guide your agent's behavior.

#### Optional features

You can also configure:

- **Voice Selection**: Choose from multiple voice options
- **Language**: Set primary language for interactions
- **Speech Recognition**: Adjust speech recognition settings
- **Call Flow**: Define custom call handling logic
- **Fallback Options**: Set backup responses or human handoff
- **Recording**: Enable/disable call recording
- **Analytics**: Configure tracking metrics
- **Enable VoiceMail Detection**: This allows the agent to detect if it reaches a voicemail and hang up or leave a message, instead of waiting for a live person. This is useful for outbound campaigns or scenarios where reaching a live person is preferred.

### 4. Add a prompt

Create a detailed prompt to guide your agent's behavior:

- Define the agent's role and personality
- Specify how to handle different scenarios
- Include any industry-specific knowledge
- Set response guidelines and restrictions
- Add example conversations or templates

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/5.jpg" alt="AI Agent dashboard" />

### 5. Test your agent

Make a test call to ensure your Voice AI Agent:

- Responds appropriately to various inputs
- Follows the defined prompt guidelines
- Handles different scenarios correctly
- Maintains natural conversation flow
