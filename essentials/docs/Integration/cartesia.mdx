## Overview

This guide explains how to link your Cartesia AI account to ContactSwing and use it when creating agents and workflows.

Learn how to connect and configure Cartesia-AI with ContactSwing. This guide shows you how to set up and use Cartesia-AI integration effectively.

# 🎙️ Cartesia-AI Integrations

## Bring your Cartesia AI account API keys to create AI Agents 🤖

Connect your Cartesia AI account to unlock advanced AI capabilities for your agents. By integrating Cartesia, you can use your API keys to create custom AI Agents directly from the dashboard. 🚀

# Step-by-step guide

Follow these steps to add the Cartesia AI integration:

### Step 1: Go to the Integrations section
In your application, go to the **Integrations** section.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/04.jpg" alt="Integrations" />

### Step 2: Select Cartesia and connect your API key
Click the **Cartesia AI** integration option to begin connecting your API key. Once you connect your Cartesia account, you can use it while creating AI Agents.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/integra2.jpg" alt="Cartesia integration" />

### Step 3: Complete the integration
After you enter your API key and email, the integration is complete. You can now create custom AI Agents directly from the Create AI Agents dashboard. ✨
