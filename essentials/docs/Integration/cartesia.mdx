# Step-by-step guide

Follow these steps to add the Cartesia AI integration:

### Step 1: Go to the Integrations section
In your application, go to the **Integrations** section.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/04.jpg" alt="Integrations" />

### Step 2: Select Cartesia and connect your API key
Click the **Cartesia AI** integration option to begin connecting your API key. Once you connect your Cartesia account, you can use it while creating AI Agents.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/integra2.jpg" alt="Cartesia integration" />

### Step 3: Complete the integration
After you enter your API key and email, the integration is complete. You can now create custom AI Agents directly from the Create AI Agents dashboard. âœ¨
