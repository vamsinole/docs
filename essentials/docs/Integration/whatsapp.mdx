## Key features

### 1. Direct messaging via actions
Send WhatsApp messages directly using **ContactsWing Actions**:
- Trigger messages based on user actions or events.
- Use pre-approved templates or fully custom messages.
- Great for reminders, updates, confirmations, and alerts.
- Fully integrated in the **Actions** section of the dashboard.

### 2. AI-powered WhatsApp agents
Create smart AI agents that respond automatically to WhatsApp messages:

## Benefits of WhatsApp integration

### 1. Enhanced customer experience
WhatsApp is a widely used platform, ensuring your messages reach customers where they're most responsive.
- Much higher open and reply rates than SMS or email.
- Personalized AI-driven conversation experience.

### 2. 24/7 AI availability
AI agents operate round-the-clock, improving response times and reducing manual load.

### 3. Scalable conversations
- Manage large volumes of chats simultaneously.
- Smooth handover to live agents when needed.

### 4. Centralized control
Manage all WhatsApp activity from one dashboard:
- Track conversations and message status.
- Monitor agent performance.
- Continuously optimize workflows and responses.

3. Configure messages or agent prompts using easy-to-use editors.
