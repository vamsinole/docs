# Step-by-step guide

Follow these steps to add the PlayHT integration:

### Step 1: Go to the Integrations section
Go to the **Integrations** section in your application. Here, you find a list of external services available for integration. Select **PlayHT** to start the setup process.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/04.jpg" alt="Integrations" />

### Step 2: Enter your PlayHT API key and user ID
To integrate PlayHT, provide the following details:
- **API Key**: Retrieve your API key from your PlayHT account settings.
- **User ID**: Enter the User ID associated with your PlayHT account.

After entering these details, click **Connect** to proceed.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/playht.jpg" alt="PlayHT API" />

### Step 3: Authorize the integration
Once you enter your API key and User ID, the system verifies your credentials. Upon successful authorization, your PlayHT account connects, enabling voice synthesis capabilities.
