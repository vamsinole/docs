# Step-by-step guide

Follow these steps to add the Sendgrid integration:

### Step 1: Go to the Integrations section
In your application, go to the **Integrations** section.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/integra.jpg" alt="Integrations" />


### Step 2: Select Sendgrid and connect your API key and from email
Click the **Sendgrid** integration option to begin connecting your API key and email. This integration lets you send campaigns from our dashboard directly.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/send.jpg" alt="Sendgrid API" />
