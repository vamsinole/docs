# Step-by-step guide

Follow these steps to add the JobRobotix integration:

### Step 1: Go to the Integrations section
Go to the **Integrations** section in your application. Here, you find a list of external services available for integration. Select **JobRobotix** to begin the setup process.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/04.jpg" alt="Integrations" />

### Step 2: Enter your JobRobotix API key and description
To integrate JobRobotix, provide the following details:
- **API Key**: Retrieve your API key from your JobRobotix account settings.
- **Description**: Add a brief description to identify the purpose of this integration (optional).

Once you fill out these fields, click **Connect** to proceed.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/job.jpg" alt="JobRobotix API" />

### Step 3: Authorize the integration
The system verifies your API key. Upon successful verification, your JobRobotix account connects, enabling seamless access to job data.

### Step 4: Access job data
Once integrated, you can access job-related data directly from the Contactswing platform. To get started:
- Navigate to the **Jobs** section in the application.
- Sync or import job data from your JobRobotix account.
- Review and manage job listings efficiently within the platform.

### Step 5: Manage job listings
After the integration, you can create, update, or manage job listings in real time. Use the centralized platform to enhance your recruitment workflow.
