# Step-by-step guide

Follow these steps to add the ElevenLabs integration:

### Step 1: Go to the Integrations section
In your application, go to the **Integrations** section, where you can connect to various external services. This lets you enhance the voices of your AI Agents. A list of available integrations appears for selection.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/11.jpg" alt="Integrations" />

### Step 2: Select ElevenLabs and connect your API key
Click the **ElevenLabs** integration option to begin connecting your API key. This integration lets you use ElevenLabs' voice options when creating your Agents.
- Once connected, you can select custom voices from ElevenLabs during the agent creation process.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/labs.jpg" alt="Labs" />
