## Step-by-step guide

Follow these steps to create your SMS action:

### 1. Select SMS action

Choose **SMS Action** from the available action types to get started.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/at.jpg" alt="Action Type" />

### 2. Configure your SMS action

Fill in the required fields:
- **Name**: Enter a name for your action (e.g., "My SMS Action").
- **Description**: Briefly describe your SMS action (e.g., "Action for sending automated SMS").
- **From**: Select the sender's phone number from the dropdown.
- **To**: Input the recipient's phone number (e.g., "123-456-7890").
- **CC**: Add any phone numbers to CC (e.g., "987-654-3210").
- **Message**: Write the message body (e.g., "Hello, this is a reminder for our meeting on [date]. Please be prepared.").

After filling in all fields, click **Create Action** to save your SMS action. ðŸŽ‰
