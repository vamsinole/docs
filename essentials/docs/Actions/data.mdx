## Step-by-step guide

Follow these steps to create your data extraction action:

### 1. Select data extraction action

Choose **Data Extraction Action** from the available action types to get started.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/at.jpg" alt="Data Extraction Action" />

### 2. Configure your data extraction action

Fill in the required fields:
- **Name**: Enter a name for your action (e.g., "My Data Extraction Action").
- **Description**: Briefly describe your action.
- **Action Timing**: Specify when the action should run (e.g., Before the call).

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/de.jpg" alt="Data Extraction Example" />

After filling in all fields, click **Create Action** to save your data extraction action. ðŸŽ‰

By following these steps, you can efficiently extract and manage data during calls in your application.
