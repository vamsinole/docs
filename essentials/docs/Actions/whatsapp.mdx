## Step-by-step guide

Follow these steps to create your WhatsApp action:

### 1. Select WhatsApp action

Choose **WhatsApp Action** from the available action types to get started.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/wa1.jpg" alt="WhatsApp Action" />

### 2. Configure your WhatsApp action

Fill in the required fields:
- **Name**: Enter a name for your action (e.g., "Send WhatsApp Message During Call").
- **From Number**: Select the registered WhatsApp number to send from.
- **To Number**: Enter the customer's phone number to send the message to.
- **Templates**: Select the template to use for the message.
- **Ensure WhatsApp Integration is Active**: Make sure your WhatsApp account is connected in the Integrations section before proceeding.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/wa2.jpg" alt="WhatsApp Integration" />

### 3. Create the action

After completing all fields and verifying the integration, click **Create Action** to save your WhatsApp action. ðŸŽ‰
