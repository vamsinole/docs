## Step-by-step guide

Follow these steps to create your Aircall action:

### 1. Select Aircall action

Choose **Aircall Action** from the available action types to get started.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/a1.jpg" alt="Aircall Action" />

### 2. Configure your Aircall action

Fill in the required fields:
- **Name**: Enter a name for your action (e.g., "Aircall Follow-up").
- **Description**: Briefly describe what this action does.
- **From and To date & time**: Set the start and end date/time for when this action should be active.
- **Topic**: Enter the relevant topic name.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/a2.jpg" alt="Aircall Action" />

### 3. Create the action

After entering all details, click **Create Action** to save and activate your Aircall action. ðŸŽ‰
