title: General Settings
---

Learn how to configure and customize your ContactSwing environment. This guide shows you how to manage general settings effectively.

## Step-by-step guide

Follow these steps to manage your settings:

### 1. Log in to the dashboard
Log in and go to the **Settings** tab.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set1.jpg" alt="Settings Dashboard" />

### 2. Update personal information
Edit your profile details:
- First name
- Last name
- Language
- Address
- Email
- Phone number

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set10.jpg" alt="Personal Info" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/personal.jpg" alt="Personal Details" />

### 3. Update login credentials
Change your account password as needed.

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/login.jpg" alt="Login" />

### 4. Manage users and profiles
Go to **Users and Controls** to add or edit users and assign profiles.

#### Team
- View all users with their name, email, and profile
- Edit or delete users
- Add a new user by providing:
  - First name
  - Last name
  - Email
  - Password
  - Phone number
  - Assigned profile

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set9.jpg" alt="Add User" />

#### Roles & permissions
- View all profiles and their descriptions
- Create new profiles by specifying:
  - Profile name
  - Description
  - Permissions for features (Knowledge Base, Calls, Actions, Customers, Emails, Orders, API Setup, workflows, campaigns, data uploads, etc.)

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set8.jpg" alt="Profiles" />

### 5. Create a user
To add a new user:
- Enter their details and assign a profile
- Refer to the **Profiles** section to define and assign profiles

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set2.jpg" alt="Create User" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set4.jpg" alt="Assign Profile" />

### 6. Create user profiles
To create a new profile:
- Enter profile name and description
- Assign permissions for features (Knowledge Base, Calls, Actions, Customers, Emails, Orders, API Setup, workflows, campaigns, data uploads, etc.)

<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set5.jpg" alt="Profile Details" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/set6.jpg" alt="Profile Permissions" />
<img src="https://storage.googleapis.com/helpsessionvideos/Contactswing/images/dev.jpg" alt="Profile Example" />

In the **Billing** section, review and update your subscription plans and monitor usage.
- Upgrade your subscription for annual savings
