---
title: Create Knowledge Base
---

**POST** `/v2/_knowledge_base`

Create a new knowledge base.

**Request Body:**
- JSON: KnowledgeBaseRequest (see OpenAPI spec)

**Response:**
- 201: Knowledge base created successfully
- 400: Invalid input
