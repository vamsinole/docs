# Make Call

Initiate an outbound call to a phone number using the ContactSwing API.

## Endpoint

```http
POST /api/v1/calls
```

## Headers

| Header | Value | Required |
|--------|-------|----------|
| `Authorization` | `Bearer {YOUR_API_KEY}` | Yes |
| `Content-Type` | `application/json` | Yes |

## Request Body

```json
{
  "phone_number": "+1234567890",
  "agent_id": "agent_12345",
  "customer_name": "John Doe",
  "customer_email": "john@example.com",
  "call_purpose": "Follow up",
  "priority": "high",
  "scheduled_time": "2024-01-15T10:00:00Z"
}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `phone_number` | string | Yes | The phone number to call (E.164 format) |
| `agent_id` | string | Yes | ID of the AI agent to handle the call |
| `customer_name` | string | No | Name of the person being called |
| `customer_email` | string | No | Email address of the person being called |
| `call_purpose` | string | No | Purpose or reason for the call |
| `priority` | string | No | Call priority (low, medium, high) |
| `scheduled_time` | string | No | ISO 8601 timestamp for scheduled calls |

## Response

### Success Response (200)

```json
{
  "success": true,
  "data": {
    "call_id": "call_67890",
    "status": "initiated",
    "phone_number": "+1234567890",
    "agent_id": "agent_12345",
    "created_at": "2024-01-15T09:55:00Z",
    "estimated_start_time": "2024-01-15T10:00:00Z"
  },
  "message": "Call initiated successfully"
}
```

### Error Response (400)

```json
{
  "success": false,
  "error": {
    "code": "INVALID_PHONE_NUMBER",
    "message": "Invalid phone number format",
    "details": "Phone number must be in E.164 format"
  }
}
```

## Example Usage

### cURL

```bash
curl -X POST https://api.contactswing.ai/api/v1/calls \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number": "+1234567890",
    "agent_id": "agent_12345",
    "customer_name": "John Doe",
    "call_purpose": "Follow up call"
  }'
```

### JavaScript

```javascript
const response = await fetch('https://api.contactswing.ai/api/v1/calls', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    phone_number: '+1234567890',
    agent_id: 'agent_12345',
    customer_name: 'John Doe',
    call_purpose: 'Follow up call'
  })
});

const result = await response.json();
console.log(result);
```

### Python

```python
import requests

url = "https://api.contactswing.ai/api/v1/calls"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "phone_number": "+1234567890",
    "agent_id": "agent_12345",
    "customer_name": "John Doe",
    "call_purpose": "Follow up call"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```

## Call Status

After initiating a call, you can check its status using the [Get Call by ID](/api-reference/endpoint/calls-get-by-id) endpoint.

## Rate Limits

- **Free Plan**: 10 calls per hour
- **Pro Plan**: 100 calls per hour
- **Enterprise Plan**: 1000 calls per hour

## Notes

- Phone numbers must be in E.164 format (e.g., +1234567890)
- Calls are queued and processed based on agent availability
- Scheduled calls can be set up to 30 days in advance
- All calls are recorded and can be accessed via the [Inbox](/essentials/docs/Inbox/call) interface
