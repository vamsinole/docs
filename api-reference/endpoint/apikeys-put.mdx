---
title: Update API Key
---

**PUT** `/v2/_apikeys/{id}`

Update an API key by ID.

**Path Parameter:**
- `id` (integer, required)

**Request Body:**
- JSON: ApiKeyRequest (see OpenAPI spec)

**Response:**
- 200: API key updated successfully
