---
title: Get Knowledge Bases
---

**GET** `/v2/_knowledge_base`

Returns all knowledge bases.

**Query Parameters:**
- `page` (integer, default: 1)
- `per_page` (integer, default: 10)

**Response:**
- 200: `{ data: [object], pagination: PaginationInfo }`
