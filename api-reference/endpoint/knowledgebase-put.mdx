---
title: Update Knowledge Base
---

**PUT** `/v2/_knowledgebase/{id}`

Update a knowledge base by ID.

**Path Parameter:**
- `id` (string, required)

**Request Body:**
- multipart/form-data: name, files, urls, faqs, uploaded_files

**Response:**
- 200: Knowledge base updated successfully
