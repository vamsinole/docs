---
title: Update Assistant
---

**PUT** `/v2/_assistants/{assistant_id}`

Update an assistant by ID.

**Path Parameter:**
- `assistant_id` (string, required)

**Request Body:**
- JSON: AssistantRequest (see OpenAPI spec)

**Response:**
- 200: Assistant updated successfully
