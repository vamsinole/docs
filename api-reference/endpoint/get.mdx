---
title: Get Actions
description: List all actions via the ContactSwing API.
---

**GET** `/v2/_actions`

Returns all actions.

**Query Parameters:**
- `page` (integer, default: 1)
- `per_page` (integer, default: 10)

**Response:**
- 200: `{ data: [object], pagination: PaginationInfo }`
- See schema in OpenAPI spec.
