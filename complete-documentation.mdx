## Authentication

All API endpoints are authenticated using a Workspace API key, provided in the `X-Workspace-Key` header. This is defined in the OpenAPI specification as an API key security scheme.

```json
"security": [
  {
    "WorkspaceAuth": []
  }
]
```

The header must be included in every request:

```http
X-Workspace-Key: <your-workspace-api-key>
```

## API Schema Reference

### Common Schemas

#### Error
```json
{
  "type": "object",
  "properties": {
    "message": {"type": "string"},
    "code": {"type": "integer"}
  }
}
```

#### Success
```json
{
  "type": "object",
  "properties": {
    "message": {"type": "string"},
    "data": {"type": "object"},
    "code": {"type": "integer"}
  }
}
```

#### PaginationInfo
```json
{
  "type": "object",
  "properties": {
    "page": {"type": "integer"},
    "per_page": {"type": "integer"},
    "total": {"type": "integer"},
    "total_pages": {"type": "integer"}
  }
}
```

### Request Schemas

#### ActionRequest
```json
{
  "type": "object",
  "properties": {
```
